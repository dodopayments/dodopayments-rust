# Rust SDK

A Rust client library generated by Stainless.

## Installation

```sh
cargo add <package-name>
```

## Usage

```rust
use sdk::{Client, ClientConfig};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let config = ClientConfig::new("https://api.example.com").with_api_key("my-key");
    let client = Client::new(config)?;
    println!("{}", client.base_url());
    Ok(())
}
```
